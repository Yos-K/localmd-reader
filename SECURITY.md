# Security Policy

MdLite Reader minimizes permissions and avoids network access in the initial version.

## Reporting

During private development, report security issues directly to the maintainer.

## Security Principles

- No network permission in the initial version
- No ads
- No analytics
- No automatic crash reporting
- JavaScript disabled in WebView
- Raw HTML is escaped
- No personal information in logs
- No production signing keys in the repository
