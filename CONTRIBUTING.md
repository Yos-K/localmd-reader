# Contributing

LocalMD Reader is in private early development.

## Development Rules

- Keep `main` buildable
- Keep commits small
- Use Conventional Commits style
- Do not commit secrets
- Do not commit production signing keys
- Do not add dependencies without a clear L0-L5 rationale
- Do not log personal information
- Review `docs/adr/README.md` before changing features or behavior
- Add exactly one `ADR-Review:` trailer to every `feat:` and `fix:` commit
- Record structural changes prompted by feedback in an ADR before implementation
