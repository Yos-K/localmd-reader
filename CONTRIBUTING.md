# Contributing

MdLite Reader is in private early development.

## Development Rules

- Keep `main` buildable
- Keep commits small
- Use Conventional Commits style
- Do not commit secrets
- Do not commit production signing keys
- Do not add dependencies without a clear L0-L5 rationale
- Do not log personal information
