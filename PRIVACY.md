# Privacy Policy

MdLite Reader is designed to work locally.

## Data Collection

MdLite Reader does not collect personal information.

## Network

The initial version does not request the Android `INTERNET` permission.

## Files

The app reads files selected by the user. Markdown content is not uploaded by the app.

## Logs

Logs must not include Markdown content, file names, file paths, full Android URIs, account data, tokens, or other personal information.
