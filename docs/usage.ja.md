# 使い方

このガイドは LocalMD Reader v0.1.0 の使い方を説明します。

## ファイルを開く

アプリ内から Markdown を開く:

1. `メニュー` をタップする。
2. `ファイルを開く` をタップする。
3. `.md` または `.markdown` ファイルを選択する。

Android から Markdown を開く:

1. ファイルマネージャを開く。
2. `.md` または `.markdown` ファイルを選択する。
3. Android が開くアプリを確認したら LocalMD Reader を選択する。

## Termux から開く

LocalMD Reader は Termux から Markdown ファイルを直接開けます。

```sh
mdlite-reader README.md
```

複数ファイルをタブとして開く:

```sh
mdlite-reader README.md docs/usage.ja.md
```

Termux コマンドは、Termux 側で Markdown 本文を読み取り、LocalMD Reader に渡します。
これにより、Android の広いストレージ権限を要求せずに Termux ホーム配下の
ファイルを開けます。

詳しいセットアップメモは `docs/termux-command.ja.md` にあります。

## タブ

- 別のファイルを開くとタブが追加される。
- タブをタップすると表示中の文書が切り替わる。
- 閉じる操作でそのタブを閉じる。
- 最後の文書タブを閉じるとウェルカム画面に戻る。
- アプリ再起動後、読み取り可能な文書のタブは復元される。

## 読書用の操作

- 文書上でピンチするとフォントサイズを変更できる。
- `メニュー` -> `ダークテーマ` または `ライトテーマ` でテーマを切り替える。
- `メニュー` -> `操作バーを下に移動` または `操作バーを上に移動` で操作部品の
  位置を変更する。
- `メニュー` -> `Switch to English` または `日本語に切り替え` でアプリ内の
  表示言語を切り替える。

## 最近開いたファイル

- `メニュー` -> `最近開いたファイル` から最近開いた文書を開ける。
- 最近開いたファイルは最大 5 件。
- 最近開いたファイルのダイアログで `履歴をクリア` を選ぶとローカル履歴を削除する。

## プライバシー

`メニュー` -> `プライバシー` からアプリ内のプライバシー説明を確認できます。

LocalMD Reader v0.1.0 には、広告、解析 SDK、ログイン、同期、クラッシュ送信、
ネットワーク権限はありません。

## 制限

LocalMD Reader v0.1.0 はビューアであり、エディタではありません。

v0.1.0 では非対応:

- Markdown 編集。
- 完全な CommonMark 互換。
- Mermaid。
- 数式。
- 脚注。
- リモート画像読み込み。
- 相対画像表示。
- クラウド同期。
