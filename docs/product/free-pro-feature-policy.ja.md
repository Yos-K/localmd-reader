# Free / Pro 機能方針

> ステータス: Free/Pro 境界の**現行の正典**。歴史的経緯は
> [`pro-development-context.ja.md`](pro-development-context.ja.md)、ドメイン規則は
> [`domain-glossary.md`](../domain/domain-glossary.md) を参照。

この文書は LocalMD Reader の Free / Pro 機能振り分けを定義します。

## 原則

- Free はローカル Markdown を読む用途で実用可能な状態を保つ。
- Pro は毎日使う人の読みやすさと操作効率を高める。
- Pro 導線は、まず Free の価値が成立していることを示し、その上で長い文書や関連する
  プロジェクト文書を快適に読むための上乗せとして説明する。
- Pro によって、プライバシー、オフライン利用、広告なし方針を弱めない。
- Pro 判定は Markdown を開いて読む中心経路から分離する。
- 支援購入は Pro 以上の追加機能階層を作らず、純粋な支援として扱う。

## Free 機能

Free は中心となるビューア機能を含みます。

- `.md` / `.markdown` ファイルを開く。
- Android ファイルマネージャから開く。
- 選択したフォルダー内の Markdown ファイルを選ぶ。
- Termux から 1 つ以上のファイルを開く。
- 一般的な Markdown テキストを表示する。
- 見出し、段落、リスト、チェックリスト、引用、コードブロック、インライン
  コード、リンク、テーブル、水平線。
- 対応言語のコードブロックをシンタックスハイライトする。
- 基本的な Mermaid 図をローカルで表示する。
- ライトテーマ。
- ダークテーマ。
- ピンチによるフォントサイズ変更。
- ダブルタップショートカット。
- タブ。
- タブ復元。
- 最近開いたファイル 5 件。
- 最近開いたファイルの履歴クリア。
- 操作部品の上部/下部配置。
- 英語/日本語 UI。
- プライバシーダイアログ。

Free を意図的に不便にしません。課金しなくても Markdown を快適に読める状態を
維持します。

## Pro 機能

Pro は買い切りで、読みやすさとパワーユーザー向けの作業効率を追加します。
意図的に弱くした Free の制限解除ではなく、長い文書、横に広いテーブル、相互リンクした
ローカル Markdown 群をより速く読むための快適化として提示します。

初期 Pro 候補:

- UIテーマ追加。
- ダブルタップ以外の高度なジェスチャーショートカット。
- テーブル閲覧強化。
- 最近開いたファイル 5 件超。
- 目次。
- 見出しジャンプ。
- 相対リンク対応。
- 相対画像対応。
- エクスポート機能。

## 今回の候補の振り分け

UIテーマ追加:

```text
Pro
```

理由: ライト/ダークテーマは Free に残します。セピア、AMOLED、高コントラスト、
アクセントカラー変更などは、頻繁に使う人向けの快適化です。

Mermaid 対応:

```text
Free basic / Pro advanced
```

理由: Mermaid は技術系 Markdown を読む主な理由になり得ます。基本的なオフライン
表示は Free に含め、図に特化した操作や出力などの快適化を将来 Pro 候補にします。

コードハイライト:

```text
Free
```

理由: コードの読みやすさは Markdown ビューアの中心価値です。基本的なハイライトは
Free に含めます。

ジェスチャーショートカット登録:

```text
Free double tap / Pro advanced
```

理由: ダブルタップはシンプルな利便機能として Free に残します。円、方向、カスタム
描画ジェスチャーはパワーユーザー向け設定です。

初期公開 Pro カタログの順序:

1. 読書環境に合わせやすい追加テーマ。
2. 追加ジェスチャーショートカット。
3. 長い文書を速く移動するためのナビゲーション。
4. 見出しジャンプショートカット。
5. 横に広いテーブルを読みやすくする強化。
6. 最近開いたファイルの拡張。
7. 関連するプロジェクト文書のリンク。
8. プロジェクト文書内のローカル画像。
9. 表示中の文書をHTMLとしてエクスポート。

## 各領域の Free ベースライン

テーマ:

- Free はライト/ダークを提供する。
- Pro は追加テーマと将来のテーマカスタマイズを提供する。

Mermaid:

- Free は、安全でオフラインなレンダラを使える場合に Mermaid 図を表示する。
- Pro は将来、図に特化した移動、調整、出力などの快適化を追加してよい。
- 同梱する Mermaid レンダラは、上流のライセンス表記を保持する。
- Mermaid 対応は Mermaid 互換の表示機能として説明し、公式の Mermaid
  製品またはサービスであるように表現しない。

コード:

- Free は対応言語のコードブロックを基本的にシンタックスハイライトする。
- 未対応の言語はエスケープ済みの等幅ブロックとして読める状態にする。

ジェスチャー:

- Free は通常の読書に必要な組み込みジェスチャーを維持する。
- Free はダブルタップショートカット設定を含む。
- Pro は円、方向、カスタム描画ジェスチャーを追加する。
- Pro はショートカットに次の見出し/前の見出しへ移動するアクションを追加する。

最近開いたファイル:

- Free は最近開いたファイルを 5 件まで保持する。
- Pro は最近開いたファイルを 20 件まで拡張し、ピン留めを追加する。

フォルダーから選ぶ:

- Free はフォルダーを選択し、その中の Markdown ファイルを選んで開けるようにする。
- 表示文言はフォルダー全体をワークスペースとして開くように見せず、フォルダー内のファイルを
  選ぶ操作として表現する。
- Pro の操作文言は一行に収まる「Markdownライブラリ」とし、右端の矢印でツリーの開閉を
  表現する。Android のフォルダー選択が必ず表示されるような表現にしない。
- Pro はプロジェクト風のフォルダー閲覧を提供する。ディレクトリを Markdown ファイルより
  先に表示し、ディレクトリを選ぶとその中へ移動し、子階層では親へ戻る操作を提供する。
- Pro ライブラリは一回限りのダイアログではなく、メニュー操作の直下に常設するツリーとする。
  文書を開くと読書のためメニューだけを閉じ、再度メニューを開いたときは同じツリーから
  すぐに別の文書を選べるようにする。
- ライブラリのタイトルにはルートから現在地までのパンくずを表示する。深い階層では
  ルートと直近の階層を残して中間を省略し、スマートフォンでも操作可能な長さに抑える。
- パンくずの先頭には選択したルートフォルダーの表示名を使い、複数のプロジェクトを
  識別できるようにする。文書プロバイダーから有効な名前を取得できない場合だけ、
  ローカライズ済みの Markdown ライブラリ名を代わりに使う。
- Pro は現在のライブラリ階層をファイル名またはフォルダー名で絞り込める。絞り込みでは
  文書プロバイダーを再問い合わせせず、入力を消すと現在階層の全件を復元する。
  一致した項目はフォルダーまたは文書という元の型を維持する。
- Pro は最後に選択したプロジェクトのルートを記憶し、Android の永続 URI 権限が有効な間は
  ライブラリ操作から同じルートを開き直す。
- 権限が取り消された場合や保存 URI が無効になった場合は、ルートの記憶を破棄して
  Android のフォルダー選択へ戻し、操作不能やクラッシュにしない。
- 空の子フォルダーでも、親へ戻る操作と別のルートフォルダーを選ぶ操作を残し、
  ナビゲーションを行き止まりにしない。

テーブル:

- Free は横スクロールとスクロールヒントでモバイルでも読めるテーブルを維持する。
- Pro は横に広いテーブルを読むとき、見出し行と先頭列を固定表示する。

相対リンク:

- Free は相対 Markdown リンクのラベルを読めるテキストとして表示する。
- Pro は安全な相対 Markdown リンクをアンカーとして残し、ローカル文書セットを
  移動できる土台にする。
- 絶対 file URL、独自スキーム、ルート相対パス、プロトコル相対 URL はアンカーとして
  出力しない。

相対画像:

- Free は相対 Markdown 画像の代替テキストを読める状態に保つ。
- Pro は安全な相対画像参照を画像として表示し、近くの画像ファイルを含むローカル文書セットを
  読みやすくする。
- 絶対 file URL、独自スキーム、ルート相対パス、プロトコル相対 URL、リモート画像 URL は
  画像として出力しない。

HTMLエクスポート:

- Freeではレンダリング済み文書のエクスポートを提供しない。
- Proでは表示中のファイル文書をAndroidの文書作成画面からHTMLファイルとして保存できる。
- 出力する本文は安全化済みレンダラーの結果だけを使い、現在の閲覧テーマと文字サイズを反映する。
- Proでは表示中のファイル文書をAndroid標準印刷画面へ渡し、対応端末では独自PDFエンジンを
  追加せず「PDFとして保存」を選択できる。
- 保存済みファイルと一時的なユーザーMarkdownはエクスポート・印刷できる。生成されたウェルカム文書は
  文書出力の型に含めず、対象外とする。選択できる出力先は端末の印刷サービスに依存する。

## Pro 機能の安全要件

- Free / Pro の境界は UI コード内の散らばった boolean ではなく、明示的な
  ドメインオブジェクトで表現する。
- 現行リリースの entitlement は一箇所で定義し、将来の Billing 連携で差し替える
  ときに Markdown 閲覧経路を変更しない。
- Pro 機能に広告、トラッキング、ログイン、解析を要求しない。
- Pro 機能は、明示的に外部データが必要な機能を除き、オフラインで動作する。
- Mermaid 対応で任意のネットワーク読み込みを有効にしない。
- Mermaid 対応では同梱したローカルコードのみを使い、CDN からレンダラを取得しない。
- コードハイライトはローカルのハイライターを使う。
- ジェスチャーショートカットには安全なデフォルトと簡単なリセットを用意する。
- Pro 権利確認ができない場合でも、ビューアとして使える状態を保つ。

## 後で決めること

Pro 実装前に次を決めます。

- Pro 価格を 99 円にするか、Play Console で設定可能な最低価格にするか。
- 支援購入を Pro と同時に追加するか。
- Pro 権利をオフライン用にどうキャッシュするか。
- Mermaid 表示を将来より小さいレンダラへ移せるかどうか。
- 最初の Pro でどのジェスチャーをカスタマイズ可能にするか。
