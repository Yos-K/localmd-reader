# リリースノート v0.1.0

MdLite Reader v0.1.0 は、最初の公開リリース候補です。

## ハイライト

- ローカルの `.md` / `.markdown` ファイルを開けます。
- Android のファイルマネージャから Markdown ファイルを開けます。
- Termux から `mdlite-reader` で 1 つ以上の Markdown ファイルを開けます。
- タブ、タブ復元、最近開いたファイル、ダークモード、ピンチによる文字サイズ
  変更に対応しています。
- 見出し、段落、リスト、チェックリスト、引用、コードブロック、インライン
  コード、リンク、テーブル、水平線をローカルで表示します。
- 広告、解析、ログイン、同期、ネットワーク権限はありません。

## 注意

- MdLite Reader v0.1.0 はビューアであり、エディタではありません。
- 10 MB を超えるファイルは、理由を示して開かずに止めます。
- リモート画像読み込み、相対画像表示、Mermaid、数式、脚注、クラウド同期、
  PDF エクスポート、HTML エクスポートはこのバージョンには含まれません。

## Play Console リリースノート

```text
MdLite Reader の初回リリースです。ローカル Markdown を読むための軽量ビューアです。

- .md / .markdown ファイルを開けます
- Android ファイルマネージャまたは Termux から開けます
- タブ、最近開いたファイル、ダークモード、ピンチ文字サイズ変更に対応
- テーブル、コードブロック、チェックリスト、リンクなどの基本Markdownを表示
- 広告、解析、ログイン、同期、ネットワーク権限なし
```
