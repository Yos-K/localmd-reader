# LocalMD Reader v0.1.2 Release Notes

## Summary

LocalMD Reader v0.1.2 is a free Play release maintenance update focused on
clearer Free/Pro guidance and production access preparation.

## Changes

- Updated the Free/Pro guidance so the free reader remains positioned as a
  complete offline Markdown reader.
- Clarified that Pro adds convenience features for heavier reading workflows,
  such as long documents and linked notes.
- Updated testing and production access notes to match the current free feature
  set and the feedback-driven changes already shipped through closed testing.
