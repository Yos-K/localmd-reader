# LocalMD Reader v0.1.2 リリースノート

## 概要

LocalMD Reader v0.1.2 は、Free/Pro の説明を分かりやすくし、本番環境
アクセス申請に向けた情報を整理するためのフリー版Playリリース向け
メンテナンス更新です。

## 変更内容

- フリー版を、オフラインで完結するMarkdownリーダーとして使える位置づけに
  整理しました。
- Pro版は、長い文書やリンクされたノートを扱うときの快適さを高める機能として
  説明するようにしました。
- クローズドテストで得たフィードバックと現在のフリー版機能に合わせて、
  テスト用テンプレートと本番環境アクセス申請用メモを更新しました。

## 確認

- ドキュメント同期チェック。
- テストスメルチェック。
- リリース前検証。
