# LocalMD Reader v0.2.10 リリースノート

## 概要

LocalMD Reader v0.2.10 は、フリー版リーダーのファイル移動、ダイアログ操作、
ジェスチャー認識の安定性を改善します。

## 変更内容

- 最近開いたファイルとピン留めファイルのダイアログに、明示的な閉じる操作を追加しました。
- Markdown ライブラリでフォルダーを開いたあと、別のフォルダーを選び直せるようにしました。
- 通常利用者向けメニューからクリップボード診断を非表示にしました。
- ダブルタップショートカットが初回タッチから安定して認識されるようにしました。
- ジェスチャースモーク失敗時と Play アップロード失敗時の GitHub Actions 診断を改善しました。
