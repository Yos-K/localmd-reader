# LocalMD Reader v0.2.11

リリース日: 2026-07-22

## 主な変更

- Android 16（API 36）を対象とするよう更新しました。
- 開いているタブと文書描画セッションの整合性を改善しました。
- ローカルMarkdownをオフライン、軽量、広告なしで読む方針を維持しています。

## 互換性

- 最小対応バージョンは引き続きAndroid 6.0（API 23）です。
- Edge-to-edgeのシステムバー処理とAndroid 16端末での動作を確認しました。
- 新しい権限、トラッキング、アカウント、ネットワークアクセスは追加していません。

## 検証

- API 36構成でFree版とPro Preview版のユニットテストが成功しました。
- CIでGradle lint、デバッグビルド、リリースBundle生成、Android 16 device smokeが成功しました。
