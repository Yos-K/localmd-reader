# LocalMD Reader v0.1.1 リリースノート

## 概要

LocalMD Reader v0.1.1 は、フリー版Playリリース向けのメンテナンス更新です。あわせて、フリー版の
Play表示では訴求しない Pro 向け内部ナビゲーション改善も含みます。

## 変更内容

- v0.1.1 用のPlayリリースメタデータを更新しました。
- フリー版Playリリースノートは、フリー版で見えるメンテナンス更新に絞りました。
- Pro向けの相対Markdownナビゲーションを内部的に強化しました:
  近くの `.md` / `.markdown` ファイル、親フォルダへのパス、見出しフラグメント、`+` を含むファイル名、
  ローカルルートによる安全確認を扱います。

## 確認

- ユニットテスト。
- リリース前検証。
