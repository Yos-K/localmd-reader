# LocalMD Reader v0.1.1 Release Notes

## Summary

LocalMD Reader v0.1.1 is a free Play release maintenance update. It also
contains Pro-gated internal navigation improvements that are not advertised as
free Play features.

## Changes

- Updated Play release metadata for v0.1.1.
- Kept the free Play release notes focused on free-visible maintenance work.
- Added Pro-gated relative Markdown navigation hardening:
  nearby `.md` / `.markdown` files, parent-directory paths, heading fragments,
  literal plus signs in filenames, and local-root safety checks.
