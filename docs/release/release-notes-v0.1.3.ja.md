# LocalMD Reader v0.1.3 リリースノート

## 概要

LocalMD Reader v0.1.3 は、フリー版リーダーのファイル選択導線を改善します。

## 変更内容

- Free 機能として「フォルダーから選ぶ」を追加しました。
- フォルダーを選択し、その中の Markdown ファイルを選んで開けるようにしました。
- フォルダー全体をワークスペースとして開くように見えないよう、ファイルを選ぶ操作として
  文言を整理しました。
- フォルダー内の `.md` / `.markdown` だけを候補にし、非対応ファイルを除外し、
  URI 重複を取り除き、表示名順に並べるモデルを追加しました。

## 確認

- ユニットテスト。
- テストスメルチェック。
- ドキュメント同期チェック。
