# リリースノート v0.1.0

LocalMD Reader v0.1.0 は、最初の公開リリース候補です。

## ハイライト

- ローカルの `.md` / `.markdown` ファイルを開けます。
- Android のファイルマネージャや他アプリから Markdown ファイルを開けます。
- Termux から `mdlite-reader` で 1 つ以上の Markdown ファイルを開けます。
- クリップボードのテキストから一時的な Markdown 文書を作成し、必要に応じて
  `名前を付けて保存` で保存できます。
- タブ、タブ復元、最近開いたファイル、ダークモード、ピンチによる文字サイズ
  変更に対応しています。
- 操作部品を上部または下部に配置でき、アプリ内表示は英語と日本語を切り替えられます。
- 見出し、段落、リスト、チェックリスト、引用、コードブロック、インライン
  コード、リンク、テーブル、水平線をローカルで表示します。
- コードブロックのシンタックスハイライトと基本的な Mermaid 図のローカル表示に対応しています。
- 広告、解析、ログイン、同期、ネットワーク権限はありません。

## 注意

- LocalMD Reader v0.1.0 はビューアであり、エディタではありません。
- クリップボードから作成した一時的な Markdown 文書は自動保存されません。
- 10 MB を超えるファイルは、理由を示して開かずに止めます。
- リモート画像読み込み、相対画像表示、Mermaid の高度な操作、数式、脚注、クラウド同期、
  PDF エクスポート、HTML エクスポートはこのバージョンには含まれません。

```text
- テーブル、コードブロック、チェックリスト、リンク、基本Mermaid図などを表示
- 広告、解析、ログイン、同期、ネットワーク権限なし
```
