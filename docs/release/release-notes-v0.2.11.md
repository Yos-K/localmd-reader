# LocalMD Reader v0.2.11

Release date: 2026-07-22

## Highlights

- Updated the app to target Android 16 (API 36).
- Improved open-tab and document-rendering session consistency.
- Kept local Markdown reading offline, lightweight, and ad-free.

## Compatibility

- Minimum supported Android version remains Android 6.0 (API 23).
- Edge-to-edge system bar handling and Android 16 device behavior were reviewed.
- No new permissions, tracking, account, or network access were added.

## Verification

- Free and Pro Preview unit tests passed on the API 36 build configuration.
- Gradle lint, debug builds, release bundle generation, and Android 16 device
  smoke passed in CI.
