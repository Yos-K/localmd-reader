# 本番環境アクセス申請向け改善メモ

このメモは、次回の Google Play 本番環境アクセス申請で説明するために、
クローズドテスト中およびテスト後に Free 版で確認・改善した内容を整理する。

## 改善方針

Free 版は、広告・トラッキング・ネットワーク権限なしでローカル Markdown を読む
基本体験を完成させる。Pro は Free 版を意図的に不便にした制限解除ではなく、長い
文書や関連するローカル文書群をより速く快適に読むための上乗せとして扱う。

## テスターの利用に基づく改善

### 初回利用

- ウェルカム画面の「Markdown ファイルを開く」を明確な操作ボタンとして扱うようにした。
- 端末の通知領域やナビゲーションバーと操作部品が重なりにくいよう、表示領域と操作バーを調整した。
- メニューやタブの見た目をライト/ダークテーマに合わせて調整した。

### 他アプリから開く

- WhatsApp など他アプリ内の Markdown 添付ファイルから開いた場合も、文書をタブとして追加するようにした。
- 別の Markdown ファイルを開いても、以前のタブを失わず切り替えられるようにした。
- 複数ファイル共有やファイルマネージャからの複数選択をタブとして扱えるようにした。

### 一時 Markdown 作成

- Markdown ファイルを直接ダウンロードできない場合に備え、クリップボードのテキストから一時 Markdown 文書を作成できるようにした。
- 一時 Markdown 文書は自動保存せず、ユーザーが内容を確認してから `名前を付けて保存` で保存できるようにした。
- クリップボードから開いた履歴をアプリ内ローカルに保持し、複数項目から選択して開けるようにした。

### 読みやすさ

- ピンチ操作によるフォントサイズ変更で、文書位置が先頭へ戻らないようにした。
- 検索をダイアログではなく検索バーとして表示し、文書を大きく隠さず前後の一致箇所へ移動できるようにした。
- 横に広いテーブルでは横スクロール可能であることを視覚的に分かりやすくした。
- コードブロックのハイライトと基本的な Mermaid 図表示を Free 版に含め、技術文書を読みやすくした。

### ファイル選択と履歴操作

- 最近開いたファイルとピン留めしたファイルの一覧に、削除操作とは別の「閉じる」導線を追加した。
- 選択したフォルダー内の Markdown 一覧に、同じ画面から別のフォルダーを選び直す導線を追加した。
- クリップボード診断は通常ユーザー向けの機能ではないため、通常メニューから非表示にした。
- Pro 向けの相対 Markdown リンクと相対画像について、MainActivity 経由の動作を medium テストで確認できるようにした。

### 安定性と安全性

- 10 MB を超えるファイルは開かず、スマートフォン上での応答性を保つようにした。
- 対応外ファイル、読み取り不能ファイル、大きすぎるファイルの理由を明確なメッセージで表示するようにした。
- WebView の Markdown 表示はネットワーク権限なし、広告なし、解析なし、ログインなしの方針を維持した。
- JavaScript URL や危険な HTML をそのまま実行・表示しないよう、レンダリングの安全性をテストで確認している。

### Free / Pro 導線

- Free 版の機能だけで、ローカル Markdown の基本閲覧、タブ、最近開いたファイル、検索、コードハイライト、基本 Mermaid 表示、クリップボード作成を使えるようにした。
- Pro 機能画面では、Free 版の価値を先に説明し、Pro は長い文書や関連文書を快適に読むための追加機能として説明するようにした。

## 申請回答で使える要約

クローズドテスト中のフィードバックを受け、初回画面の操作導線、他アプリから Markdown を開く挙動、タブ保持、一時 Markdown 作成、文書内検索、フォントサイズ変更時の表示位置保持、テーブル表示、エラーメッセージを改善しました。Free 版だけでローカル Markdown を広告・トラッキング・ネットワーク権限なしに閲覧できるようにし、Pro は長い文書や関連文書をより快適に読むための追加機能として整理しました。

## 追加で確認する項目

- Play Store からインストールした Free 版で、アプリ内から `.md` / `.markdown` を開ける。
- 他アプリから Markdown 添付ファイルを開いても、既存タブが失われない。
- クリップボードから一時 Markdown を作成し、必要な場合だけ保存できる。
- 検索バーで前後の一致箇所へ移動できる。
- ピンチでフォントサイズを変えても表示位置が大きく崩れない。
- ライト/ダークテーマで文字、タブ、メニューが読める。
- コードブロックと基本 Mermaid 図が表示される。
- プライバシーダイアログで、広告なし・解析なし・ネットワーク権限なしを確認できる。
