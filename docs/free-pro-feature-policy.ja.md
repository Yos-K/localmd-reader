# Free / Pro 機能方針

この文書は LocalMD Reader の Free / Pro 機能振り分けを定義します。

## 原則

- Free はローカル Markdown を読む用途で実用可能な状態を保つ。
- Pro は毎日使う人の読みやすさと操作効率を高める。
- Pro によって、プライバシー、オフライン利用、広告なし方針を弱めない。
- Pro 判定は Markdown を開いて読む中心経路から分離する。
- 支援購入は Pro 以上の追加機能階層を作らず、純粋な支援として扱う。

## Free 機能

Free は中心となるビューア機能を含みます。

- `.md` / `.markdown` ファイルを開く。
- Android ファイルマネージャから開く。
- Termux から 1 つ以上のファイルを開く。
- 一般的な Markdown テキストを表示する。
- 見出し、段落、リスト、チェックリスト、引用、コードブロック、インライン
  コード、リンク、テーブル、水平線。
- ライトテーマ。
- ダークテーマ。
- ピンチによるフォントサイズ変更。
- タブ。
- タブ復元。
- 最近開いたファイル 5 件。
- 最近開いたファイルの履歴クリア。
- 操作部品の上部/下部配置。
- 英語/日本語 UI。
- プライバシーダイアログ。

Free を意図的に不便にしません。課金しなくても Markdown を快適に読める状態を
維持します。

## Pro 機能

Pro は買い切りで、読みやすさとパワーユーザー向けの作業効率を追加します。

クローズドテスト準備中は、公開リポジトリには Free / Pro の境界モデルと
ドキュメントだけを置きます。Pro 機能の具体実装と Play Billing 連携は、
Free 版のリリース経路が安定してから行います。

初期 Pro 候補:

- UIテーマ追加。
- Mermaid 図の表示。
- コードブロックのシンタックスハイライト。
- ジェスチャーショートカットの登録。
- テーブル閲覧強化。
- 最近開いたファイル 5 件超。
- 目次。
- 見出しジャンプ。
- フォルダ閲覧。
- 相対リンク対応。
- 相対画像対応。
- エクスポート機能。

## 今回の候補の振り分け

UIテーマ追加:

```text
Pro
```

理由: ライト/ダークテーマは Free に残します。セピア、AMOLED、高コントラスト、
アクセントカラー変更などは、頻繁に使う人向けの快適化です。

Mermaid 対応:

```text
Pro
```

理由: 基本的な Markdown 閲覧には必須ではありません。パーサ/レンダラの複雑さも
増えるため、機能境界を分けます。

コードハイライト:

```text
Pro
```

理由: Free でもコードブロックは読める状態にします。シンタックスハイライトは
開発者向けの強化機能です。

ジェスチャーショートカット登録:

```text
Pro
```

理由: ピンチによるフォントサイズ変更など、基本読書に必要な組み込みジェスチャーは
Free に残します。タップ、ダブルタップ、長押し、スワイプなどのカスタマイズは
パワーユーザー向け設定です。

## 各領域の Free ベースライン

テーマ:

- Free はライト/ダークを提供する。
- Pro は追加テーマと将来のテーマカスタマイズを提供する。

Mermaid:

- Free では Mermaid フェンスをコードブロックとして表示してよい。
- Pro では安全でオフラインなレンダラを使える場合に図として表示する。

コード:

- Free はコードブロックを読みやすい等幅ブロックとして表示する。
- Pro はシンタックスハイライトと、将来的には言語ラベルを追加する。

ジェスチャー:

- Free は通常の読書に必要な組み込みジェスチャーを維持する。
- Pro はユーザー設定可能なショートカットを追加する。

## Pro 機能の安全要件

- Free / Pro の境界は UI コード内の散らばった boolean ではなく、明示的な
  ドメインオブジェクトで表現する。
- Pro 機能に広告、トラッキング、ログイン、解析を要求しない。
- Pro 機能は、明示的に外部データが必要な機能を除き、オフラインで動作する。
- Mermaid 対応で任意のネットワーク読み込みを有効にしない。
- コードハイライトはローカルのハイライターを使う。
- ジェスチャーショートカットには安全なデフォルトと簡単なリセットを用意する。
- Pro 権利確認ができない場合でも、ビューアとして使える状態を保つ。

## 後で決めること

Pro 実装前に次を決めます。

- Pro 価格を 99 円にするか、Play Console で設定可能な最低価格にするか。
- 支援購入を Pro と同時に追加するか。
- Pro 権利をオフライン用にどうキャッシュするか。
- Mermaid 表示を WebView、Java、将来のレンダリング部品のどこで実装するか。
- 最初の Pro でどのジェスチャーをカスタマイズ可能にするか。
