# Release Notes v0.1.0

LocalMD Reader v0.1.0 is the first public release candidate.

## Highlights

- Open local `.md` and `.markdown` files.
- Open Markdown files from Android file managers.
- Open one or more Markdown files from Termux with `mdlite-reader`.
- Read Markdown with tabs, tab restoration, recent files, dark mode, and pinch
  font sizing.
- Render common Markdown locally: headings, paragraphs, lists, checklists,
  blockquotes, code blocks, inline code, links, tables, and horizontal rules.
- Keep reading private: no ads, no analytics, no login, no sync, no network
  permission.

## Notes

- LocalMD Reader v0.1.0 is a viewer, not an editor.
- Files larger than 10 MB are rejected with a clear message.
- Remote image loading, relative image rendering, Mermaid, math, footnotes,
  cloud sync, PDF export, and HTML export are not included in this version.
