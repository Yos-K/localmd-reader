# LocalMD Reader Privacy Policy

LocalMD Reader is designed as a local-first Markdown viewer.

## Summary

- No ads
- No analytics SDK
- No account or login
- No automatic crash reporting
- No `INTERNET` permission in v0.1.0
- No upload of Markdown content by the app

## Data Collection

LocalMD Reader does not collect personal information.

The app does not collect names, email addresses, account identifiers, device
identifiers, location data, contacts, usage analytics, or advertising IDs.

## Privacy Contact

Use the developer contact email configured in Google Play Console for privacy
questions about LocalMD Reader.

## Files

LocalMD Reader reads files that the user explicitly opens through Android file
access flows.

Markdown content is rendered on the device. The app does not upload Markdown
content, file contents, file names, file paths, or Android document URIs.

## Local Storage

The app stores small local preferences in app-private storage:

- Theme setting
- Controls placement setting
- Recently opened file metadata, limited to 5 entries
- Open tab metadata for tab restoration

Recent file and tab metadata may include a display name and Android document
URI. This data stays in app-private storage and can be removed by clearing app
data. Recent file history can also be cleared from the app.

## Network

LocalMD Reader v0.1.0 does not request the Android `INTERNET` permission.

The app itself cannot send documents over the network. HTTP and HTTPS links in
Markdown are opened by handing them to another app chosen by Android.

## Logs

Application logs must not include Markdown content, file names, file paths, full
Android URIs, account data, tokens, or other personal information.

## Changes

If future versions add network access, analytics, sync, crash reporting, or
other data handling, this policy must be updated before release.

# LocalMD Reader プライバシーポリシー

LocalMD Reader は、ローカルファーストの Markdown ビューアとして設計されています。

## 概要

- 広告なし
- 解析 SDK なし
- アカウントやログインなし
- 自動クラッシュ送信なし
- v0.1.0 では `INTERNET` 権限なし
- アプリによる Markdown 本文のアップロードなし

## データ収集

LocalMD Reader は個人情報を収集しません。

氏名、メールアドレス、アカウント識別子、端末識別子、位置情報、連絡先、
利用解析、広告 ID は収集しません。

## プライバシー問い合わせ

LocalMD Reader のプライバシーに関する問い合わせには、Google Play Console に設定した
開発者連絡先メールアドレスを使用します。

## ファイル

LocalMD Reader は、ユーザーが Android のファイルアクセス機能を通じて明示的に
開いたファイルを読み取ります。

Markdown 本文は端末上でレンダリングされます。アプリは Markdown 本文、
ファイル内容、ファイル名、ファイルパス、Android document URI をアップロードしません。

## ローカル保存

アプリは小さな設定を app-private storage に保存します。

- テーマ設定
- 操作部品の配置設定
- 最近開いたファイルのメタデータ、最大 5 件
- タブ復元用の開いていたタブのメタデータ

最近開いたファイルとタブのメタデータには、表示名と Android document URI が
含まれる場合があります。このデータは app-private storage に保存されます。
アプリデータの削除で消去でき、最近開いたファイルの履歴はアプリ内からも
クリアできます。

## ネットワーク

LocalMD Reader v0.1.0 は Android の `INTERNET` 権限を要求しません。

アプリ自体は文書をネットワークへ送信できません。Markdown 内の HTTP / HTTPS
リンクは、Android が選択する別アプリへ渡して開きます。

## ログ

アプリケーションログには Markdown 本文、ファイル名、ファイルパス、Android URI 全文、
アカウント情報、トークン、その他の個人情報を含めません。

## 変更

将来のバージョンでネットワークアクセス、解析、同期、クラッシュレポート、
その他のデータ処理を追加する場合は、リリース前にこのポリシーを更新します。
