# MdLite Reader

Read local Markdown files with a calm, fast viewer.

## Everyday Notes

- Open `.md` and `.markdown` files
- Keep multiple documents in tabs
- Restore tabs after restart
- Change text size with a pinch gesture

## Code Block

```java
public final class Note {
    private final String title;
    private final String body;
}
```
