# コントリビューション

LocalMD Reader は private の開発初期段階です。

## 開発ルール

- `main` はビルド可能に保つ
- コミットは小さくする
- Conventional Commits 風にする
- 秘密情報をコミットしない
- 本番署名鍵をコミットしない
- L0-L5で説明できない依存を追加しない
- ログに個人情報を入れない
- 機能追加・修正の前に `docs/adr/README.ja.md` を確認する
- すべての `feat:` / `fix:` コミットに `ADR-Review:` トレーラを1つ付ける
- 指摘を受けて構造を変える場合、実装前にADRへ決定を記録する
